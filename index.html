What it does
From ArXiv to your inbox in four steps.
📡

Fetches from ArXiv

Pulls the latest papers for any topics you care about. Add or remove topics from Preferences at any time.

🧠

Summarizes on-device

Runs a HuggingFace model on your hardware. Works offline after the first download. No API keys, ever.

🗂

Clusters by topic

Groups related papers using embeddings and k-means so you can scan a cluster and move on.

🔔

Lives in your menu bar

Sits quietly on macOS and Linux. Notifies you when a digest is ready. Out of the way until you need it.

📅

Runs on a schedule

Set a morning or evening digest time. Sift handles the rest. Hit "Fetch Now" to run one immediately.

⚙️

Adapts to your hardware

Detects your CPU, GPU, or Apple Silicon on first run and picks the right model for your setup.

Requirements
What you need to run it.
Python 3.11+ macOS 12+ or Linux x86_64 4–16 GB disk space Internet connection for fetching papers
